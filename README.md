# EmqxInfluxExporter
Emqx plugin to export events to InfluxDB.

##### Events
* client.connected
* client.disconnected
* message.publish
* message.delivered
* message.acked
* client.subscribe
* client.unsubscribe
* session.subscribed
* session.unsubscribed

##### Environment variables
* INFLUX_URL
* INFLUX_DBNAME
